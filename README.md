# 🧪 Misión Python: Escape Lab

> Un videojuego interactivo en consola desarrollado en Python para poner en práctica los fundamentos de la programación.

---

## 📖 Descripción

**Misión Python: Escape Lab** es un pequeño videojuego desarrollado en **Python** que se ejecuta desde la consola.

El jugador despierta dentro de un laboratorio secreto sin recordar cómo llegó allí. Su misión consiste en explorar el laboratorio, obtener los objetos necesarios y descubrir el código de emergencia para abrir la puerta principal y escapar.

Este proyecto fue desarrollado como práctica de los conceptos fundamentales de Python siguiendo una metodología de aprendizaje basada en proyectos.

---

## 🎮 Historia

Has despertado dentro de un laboratorio secreto.

Las luces parpadean.

Las puertas permanecen bloqueadas.

Una voz anuncia:

> **"Protocolo de confinamiento activado."**

Tu única oportunidad será explorar el laboratorio y encontrar:

- 🔑 Una llave.
- 💳 Una tarjeta magnética.
- 🔢 El código de emergencia.

Solo entonces podrás escapar.

---

## 🎯 Objetivo

El objetivo del juego es conseguir todos los elementos necesarios para abrir la puerta principal.

Para lograrlo el jugador deberá:

- Explorar el laboratorio.
- Encontrar objetos.
- Consultar el inventario.
- Descubrir el código secreto.
- Introducir correctamente el código **314**.

---

## ⚙️ Funcionamiento

El flujo general del juego es el siguiente:

1. El jugador ingresa su nombre.
2. Selecciona el nivel de dificultad.
3. Explora el laboratorio.
4. Obtiene los objetos necesarios.
5. Consulta el inventario cuando lo desee.
6. Intenta abrir la puerta principal.
7. Si posee todos los elementos e ingresa el código correcto, logra escapar.
